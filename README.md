## AFPlayer 安防播放器

基于ijkplayer的播放器

参考与引用：

- ijkplayer:https://github.com/bilibili/ijkplayer
- 录像与截图：https://github.com/zhu528/ijkplayer-record
- 播放器UI:https://github.com/ILoveLin/IjkRecordPlayerAddUI

### 基本功能

| 功能                 | 支持 | 说明         |
| -------------------- | ---- | ------------ |
| rtsp播放，h264流播放 | √    | 部分功能修改 |
| 录像、截图           | √    | 添加功能     |
| 实时kbs统计          | √    | 添加功能     |
